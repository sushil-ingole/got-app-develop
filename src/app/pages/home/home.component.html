Welcome to the Game Of Thrones Test

Hitachi Energy 2024
Explanation:

The main objective of this test is to validate the Angular basic knowledge of the candidates by creating a component from scratch based on a simplified description of the component. The candidates are free to imagine how the component should work based on their own experience.

The evaluation will have 3 main components:
  • Code structure
  • Logic
  • Cosmetic / Style work
Task 1 : Structure
  • Using git clone repo from: repo
  • Create new branch from develop using the following notation: dev/hitachi-test-1/$your-name
  • Generate new component to show all characters from game of thrones using the documentation on the web site: www.thronesapi.com and the service provided with this repo: GotInfoService.
  • Fix any problem you might have fetching the data or create your own service.
    • Create/Update or fix the necessary functions to retrieve the information from the API.
    • Create catalog component and present all the characters in a grid view.
    • Show image and basic information like first and last name.
Task 2 : Style
  • Using Angular Material (already installed), provide style to the grid component to present the characters aligned with the GoT Theme.
  • Make it responsive and visually in line with the dark them of GoT.
  • Remember to create as many commits as you need per each action.
Task 2 : Push and PR
  • Push your changes to the remote repo.
  • Create a pull request and add Approvers: Julian Toro and David Restrepo.
